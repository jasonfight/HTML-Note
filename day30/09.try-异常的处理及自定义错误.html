异常的捕捉: try {...} catch(e){ ... } finally{...}
catch 中可以对不同类型异常进行捕捉,
eg: if (e instanceof errotype){....}
常见异常:
1.Error 所有异常的父类,可以通过 new Error 来创建一个自定义异常
2.EvalError: 关于eval的异常
3.intermalError: 引擎内部出现错误(不标准)
4.RangeError: 超过最小值或最大值
5.ReferenceError: 变量不存在
6.SyntaxError: 语法错误
7.TypeError: 类型错误
8. URLError: URL解析错误



自定义异常: eg : try{ throw new Error('自定义错误提示内容','filename',行号)}
catch(e){ consolo.log(e.message)}
